Интернет-магазин запчастей для ремонта мобильных телефонов.

## Функциональность

- Просмотр каталога товаров
- Добавление товаров в корзину
- Управление количеством товаров
- Оформление заказа
- Адаптивный дизайн

## Технологии

- HTML5
- CSS3 (Flexbox/Grid)
- JavaScript (ES6+)
- Local Storage

## Установка

1. Клонируйте репозиторий
